# 项目概述 python1176

本项目是一个基于Python深度学习的音乐推荐系统，旨在为用户提供个性化的音乐推荐服务。

# 技术栈

## 后端

- 开发语言：Python
- 框架：Django
- Python版本：3.7.7
- 数据库：MySQL 5.7+
- 数据库工具：Navicat 11+

## 前端

- Node.js
- Vue
- HTML

# 功能模块

## 系统首页

展示系统的主要功能入口。

## 音乐播放界面

- 展示歌曲信息：歌名、作者、演唱者、流派、歌词等。
- 支持功能：歌曲下载、点评、收藏。

## 音乐推荐功能

用户可通过输入歌名，获取与之相关的歌曲推荐。

## 后台管理系统

- 管理员对网站内歌曲信息进行管理。

# 系统角色

- 普通用户：登录系统，使用音乐推荐功能。
- 管理员：登录系统，进行数据管理和内容操作。

# 运行环境

- 前端：Node.js，Vue，HTML
